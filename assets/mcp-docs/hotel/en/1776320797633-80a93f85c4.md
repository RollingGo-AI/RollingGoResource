
# RollingGo MCP Server

![Version](https://img.shields.io/badge/Version-1.0.0-blue)

![MCP Protocol](https://img.shields.io/badge/MCP-Protocol-green)

![License](https://img.shields.io/badge/License-MIT-yellow)

  

**RollingGo MCP** is a high-performance hotel search and recommendation service built on the MCP (Model Context Protocol) standard. It connects Large Language Models with global hotel data, providing real-time structured hotel search capabilities for AI assistants, agents, travel planning tools, and IDE plugins like Cursor/Windsurf.

  

Simply describe your hotel search needs in natural language—such as "Find me a 5-star hotel near West Lake in Hangzhou for January 1st"—and get instant results with hotel names, prices, locations, and booking links. The service returns both basic information and deep details (amenities, distance, descriptions), significantly enhancing the interactive experience and recommendation quality of conversational travel applications.

  

> 💡 **No coding required!** Once configured, any MCP-compatible AI assistant gains hotel search capabilities.

## 🚀 Quick Start

  

### Step 1: Get Your API Key

  

1. [Apply for API-KEY](https://mcp.agentichotel.cn/apply)

2. Click "**Apply**" and fill in your basic information

3. Once approved, your **API Key** will be sent to your provided email

  

> ⚠️ Please keep your API Key secure and never share it with others

  

### Step 2: Configure in Your AI Assistant

  

#### Using in Cursor / Windsurf (Remote Mode)

  

```json

{

"mcpServers": {

"rollinggo-hotel": {

"serverUrl": "https://mcp.aigohotel.com/mcp",

"headers": {

"Authorization": "Bearer YOUR_API_KEY",

"Content-Type": "application/json"

}

}

}

}

```

  

#### Local Mode

  

Suitable when you have the `rollinggo-mcp` service running locally.

  

1. **Start the backend service**: Ensure the local service is running and listening at `http://localhost:8000/mcp`.

2. **Add configuration**: Use the traditional `url` field in `mcp.json`.

  

```json

{

"mcpServers": {

"rollinggo-hotel": {

"url": "http://localhost:8000/mcp",

"type": "http",

"headers": {

"X-Secret-Key": "YOUR_API_KEY"

}

}

}

}

```

  

### Step 3: Start Using

  

Send a request to your AI assistant, for example:

> "Find me 4-star or above hotels near West Lake in Hangzhou with lake views, within the next 3 days."

  

The AI will automatically call the tool and return the hotel list.

## Configuration Parameters

  

### Core Parameters

  
| Parameter | Required  | Description  | Example  |
|------|------|------|------|
| place | ✅ | Search location | "Hangzhou", "Disneyland" |
| placeType | ✅ | Location type | 城市、景点、机场、地铁站、区/县... |
| originQuery | ✅ | Your original request description | "Find me a hotel"  |
| checkIn | ❌ | Check-in date (yyyy-MM-dd) | 2026-05-01 |
| stayNights | ❌ | Number of nights | 2 |
| starRatings | ❌ | Star rating range| [4, 5] represents 4-5 stars |
| size | ❌ | Number of results (default: 10, max: 20) | 5 |


>  **Note:**

> - Actual fields are subject to the API response. The examples above show only partial fields.

> - As backend capabilities evolve, fields may be added or adjusted. MCP clients should handle them with a "use if available, ignore if not" approach for compatibility.

## 🔒 Security & Authentication

  

### Authentication Methods
| Environment | Method  | Header |
|------|----------|--------|
| Remote (Cloud) | Bearer Token | `Authorization: Bearer YOUR_API_KEY` |
| Local| Secret Key | `X-Secret-Key: YOUR_API_KEY` |
  

### Security Best Practices

  

- Keep your API Key secure and avoid hardcoding it in code

- Use environment variables for production environments

- Use HTTPS protocol for public network access
